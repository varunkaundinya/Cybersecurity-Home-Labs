# Cybersecurity-Home-Labs
A collection of hands-on cybersecurity projects focusing on SIEM, traffic analysis, and vulnerability management
# 🛡️ Cybersecurity Home Labs Portfolio

## About Me
I am a Master of Computer Applications (MCA) graduate actively pursuing a career as a Security Operations Center (SOC) Analyst. This repository serves as a practical portfolio of my hands-on experience in network defense, threat detection, and vulnerability management.

## Project Overview
This repository contains three core lab environments I built and documented to demonstrate practical SOC analyst skills:

1. **[SIEM Brute Force Detection](./Splunk-Brute-Force)**: Deployed Splunk Enterprise to ingest Windows Event Logs and configured alerts for Event ID 4625 to detect simulated brute-force attacks.
2. **[Network Traffic Analysis](./Wireshark-Traffic-Analysis)**: Simulated a network port scan using Nmap and analyzed the resulting PCAP data in Wireshark to identify attack signatures on the wire.
3. **[Vulnerability Management](./Nessus-Vulnerability-Scan)**: Deployed a vulnerable Linux virtual machine (Metasploitable) and conducted credentialed/non-credentialed network scans using Tenable Nessus to identify and prioritize critical CVEs.

## Core Competencies Demonstrated
- **SIEM / Log Analysis:** Splunk, Windows Event Viewer
- **Network Traffic Analysis:** Wireshark, Packet Analysis, TCP/IP Protocols
- **Vulnerability Scanning:** Tenable Nessus, CVE Identification, Patch Prioritization
- **Offensive Tooling (For Defensive Context):** Nmap, Basic Metasploit usage
